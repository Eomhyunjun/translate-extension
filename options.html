상세 설정

확장 아이콘을 클릭하면 빠른 번역 팝업이 열립니다. 상세 설정은 확장 아이콘을 우클릭해서 이 화면에서 관리합니다.

번역 설정

단축키

인라인 번역 켜기/끄기 CmdShift1
좌우 분할 번역 켜기/끄기 CmdShift2

Chrome의 확장 프로그램 단축키 설정에서 원하는 키로 바꿀 수 있습니다.

페이지에서 찾은 문단을 API에 몇 개씩 묶어서 보낼지 정합니다. 최대 50개까지 설정할 수 있고, 숫자가 크면 빠를 수 있지만 실패 가능성이 올라갑니다.

텍스트 번역

현재 번역 엔진과 언어 설정으로 붙여넣은 텍스트를 바로 번역합니다.

API 키

API 키는 동기화 저장소가 아니라 이 브라우저의 로컬 저장소에만 저장됩니다.

사용 로그

로그 저장은 기본 꺼짐입니다. 켜면 최대 100개 요청 배치를 로컬에 보관하고 입력/출력 텍스트 미리보기와 실제 또는 추정 토큰 수를 표시합니다.
